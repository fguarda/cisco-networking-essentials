# Cisco Projects Portfolio

This repository contains projects done during the Cisco Networking Academy course. The projects are created using Cisco Packet Tracer and aim to reinforce the network fundamentals learned throughout the course.

## Project Structure

Each module has a folder with the module name, which includes:
- `.pkt` file for Cisco Packet Tracer with the built topology
- `README.md` with an explanation of the project and tools used

## How to Run the Projects

1. Download and install [Cisco Packet Tracer](https://www.netacad.com/portal/resources/packet-tracer).
2. Clone this repository or download the files.
3. Open the `.pkt` files in Packet Tracer.

## Scripts Used

- **iniciar-lab.sh**: A script to automate the environment startup.
